# 🌸 Perfume Analytics Dashboard

## Overview

This project is an interactive dashboard developed using **Streamlit** for analyzing perfume data.

The dashboard provides visual insights into perfume characteristics, ratings, popularity, brands, and includes a machine learning recommendation feature.

## Features

- Interactive dashboard
- Perfume characteristics analysis
- Rating analysis
- Performance analysis
- Popularity analysis
- Brand analysis
- Correlation analysis
- Machine Learning recommendation

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Joblib

## Run the project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
